INTRO

Estoy en búsqueda de trabajo como programadora en React. Poseo sólidos conocimientos en HTML, CSS, Javascript, Typescript, React y Python. Me desenvuelvo muy bien realizando peticiones HTTP, con el manejo de JSON y la tecnología Git. Asimismo, poseo un nivel de inglés intermedio y un nivel de portugués básico.

Tengo una personalidad amigable y me encanta trabajar en equipo. Busco siempre dar un enfoque práctico a la resolución de problemas. Presto gran atención a los detalles, y tengo una gran capacidad para comprender y aprender. Poseo excelentes habilidades de comunicación verbal y escrita, así como organizativas e interpersonales.

Me desempeñé durante casi dos años en el área de juicios de un importante estudio jurídico dedicado a derecho ambiental. Mis tareas consistieron principalmente en el análisis de la prueba producida en los distintos expedientes, redacción de contestaciones de demanda, confección y presentación de escritos y comunicación con los distintos juzgados.

Work

En el corriente año participé en el frontend de una plataforma para realizar pedidos para una importante cadena de heladerías. El proyecto fue realizado con React. Mis tareas consistían en enviar peticiones a un backend y mostrar la data en tablas, tareas de internacionalización, y mucho maquetado web.

Como proyecto de práctica, cree un sitio web responsive con información sobre bandas de distintos géneros. Este proyecto fue realizado en Octubre 2021, utilizando React. Realicé un diseño y maquetado responsive. Asimismo, en este proyecto realicé peticiones axios y posee un authentication wall.

Otro proyecto académico que poseo, implicó la réplica del sitio web de la organización Atlantic Council. El sitio es responsive y fue realizado con React, utilizando hooks y props.

En Enero de 2021, como proyecto final de un curso de HTML, CSS y Javascript, creé un sitio web responsive para emprendimiento de venta de comida vegetariana utilizando las mencionadas tecnologías.

En el siguiente link se puede acceder a videos donde se muestra el funcionamiento de estas páginas. Asimismo, en mi Github pueden encontrar el código de las mismas.
